# BGM

이 폴더에 MP3 파일을 넣으면 앱 배경음악으로 사용됩니다.

## 사용법

1. MP3 파일을 이 폴더에 복사 (bgm1.mp3, bgm2.mp3, ... 자유롭게)
2. Supabase에 업로드:
   ```
   cd scripts && npm run upload-bgm
   ```
3. 앱에서 자동으로 셔플 재생됩니다.

## 규칙

- 파일명: 한글/공백 없이 영숫자+하이픈만 (`bgm1.mp3`, `calm-rain.mp3`)
- 포맷: MP3 (128kbps 이상 권장)
- 이미 업로드된 파일은 스킵됨 (중복 업로드 없음)
