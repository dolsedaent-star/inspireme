## 마주친 함정·우회

| 문제 | 우회 |
|---|---|
| EAS 무료 Android 빌드 한도 | 로컬 `expo run:android --variant release` (~16초) |
| `eas build --local` Windows 미지원 | `expo run:android --variant release` 대체 |
| Metro가 root에서 `./index.ts` 찾음 | root에 `index.js` redirect + apps/mobile main을 `.js`로 |
| EAS keystore vs 로컬 keystore signature mismatch | 첫 설치 시 `adb uninstall` 후 새 빌드 install |
| Gemini 일일 20회 quota | Google Cloud Billing 활성화 (Tier 1 무제한) |
| Storage anon write RLS deny | 0002 마이그레이션의 Storage 정책 부분만 따로 SQL Editor 실행 |
| `SUPABASE_SERVICE_ROLE_KEY` 자리에 anon 키 들어감 | JWT decode로 role 확인, 콘솔에서 정확한 키 복사 |
| RN client storage upload 안 됨 (release build) | 동적 생성은 wiki URL 그대로 저장 → server backfill |
| RN의 console.log가 release build에서 stripped | 임시 `Alert.alert` 기반 debug toggle (DEBUG 플래그) |
| 한 ↻에서 같은 candidate 3번 pick | `pickNextCandidate`에 `excludeSlugs` 추가, in-flight 추적 |
| Wikipedia 이미지 URL이 RN 기본 UA로 403 | `wikiImageSource()`가 `User-Agent` header 첨부 + Storage URL 우선 |
