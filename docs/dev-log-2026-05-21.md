# 개발 로그 — 2026-05-21

## 추가 및 수정 내역

### 1. BGM 무드 시스템
- FigureScreen 스크롤 위치에 따라 배경음악 무드 자동 전환
- 4가지 무드: `young` (요약·키워드), `hardship` (추락과 도약), `death` (말년과 죽음), `default` (나머지)
- Supabase `bgm` 버킷에서 파일명 prefix로 분류 (`young_`, `pain_`, `tension_`, `die_`, `bgm_`)
- 섹션 전환 시 fade-out 후 새 트랙 재생 (expo-av)
- 스크롤 중 무드 전환 방지 → 스크롤 멈춘 뒤 적용 (`onMomentumScrollEnd` + `onScrollEndDrag`)
- Supabase 미연결 시 번들된 `bgm1.mp3` 폴백

### 2. 위인 카드 점진적 로딩
- 기존: 3장 모두 준비된 후 한꺼번에 표시 (최대 16초 대기)
- 변경: 1장 완료될 때마다 즉시 화면에 표시 (`onIncremental` 콜백)
- 로딩 중 스피너 + 진행 텍스트 표시 ("2명 완료 — 나머지 생성 중…")

### 3. 위키피디아 갤러리 폴백
- DB에 gallery 데이터가 없는 위인의 경우 Wikipedia `page/media-list` API로 사진 자동 보완
- 로고·서명·지도 이미지 자동 제외, 커버 이미지 중복 제외
- 최대 5장, 캡션 텍스트 포함

### 4. 살아있는 인물 필터
- 사망연도 없고 1940년 이후 출생인 인물 자동 제외 (마이클 조던, 오프라 윈프리 등)
- Supabase 쿼리 필터: `death_year IS NOT NULL OR birth_year < 1940`

### 5. FigureCard UI 개선
- **시대 태그 제거**: "근대이전" 등 부정확한 AI 생성 era 태그 표시 안 함
- **얼굴 가림 완화**: scrim(어두운 그라디언트) 범위 축소
  - 상단 전체 scrim: 35% → 15% 투명도
  - 하단 밴드 높이: 70%/50%/32% → 45%/32%/22%
- **인용구 제거**: 카드에서 quote 텍스트 삭제 → 얼굴 더 크게 표시

### 6. 개인화 문구 개선 (`personalize.ts`)
- 기존: "~를 경험하였다" (죽음 이벤트에도 동일하게 적용되어 부자연스러움)
- 변경:
  - 나이 근접 (±1세): `"XXX도 N세에 '제목'의 시기를 보내고 있었습니다."`
  - 유저가 더 나이 많은 경우: `"당신의 나이보다 N살 전, XXX는 '제목'의 시기를 보내고 있었습니다."`
  - 위인이 더 나이 많은 경우: `"XXX는 지금의 당신보다 N살 후인 M세에 '제목'의 시기를 맞이했습니다."`
- 불필요한 "준비할 시간" 문장 제거

### 7. 인생 곡선 레이블
- LifeCurveChart 데이터 포인트에 `label_ko` 텍스트 표시
- 포인트마다 위/아래 교차 배치로 겹침 방지

### 8. Metro 번들러 설정
- `metro.config.js`: `ws`, `stream` 등 Node.js 빌트인 모듈을 빈 shim으로 리다이렉트
- Supabase realtime-js가 React Native에서 빌드 오류 내던 문제 해결

## 버그 수정

| 문제 | 원인 | 해결 |
|------|------|------|
| "모든 위인을 다 만났습니다" 오조기 표시 | `.not('death_year', 'is', null)` 필터로 대부분 인물 제외됨 | 해당 필터 제거 |
| BGM 음량이 나왔다 줄어들었다 반복 | `onScroll` 이벤트 200ms 스로틀로 무드 전환 남발 | scroll-stop 이벤트로 변경 |
| BGM 안드로이드에서 무음 | `setVolumeAsync` 루프 fade-in이 Android에서 실패 | 시작부터 MAX_VOL로 재생 |
| "추락과 도약" 섹션에서 무드 안 바뀜 | 2초 디바운스 중 스크롤 지나가면 타이머 취소 | 디바운스 제거, scroll-stop 방식으로 통일 |
| build 시 `stream` 모듈 오류 | 모노레포 루트에서 실행 시 `apps/mobile/metro.config.js` 미적용 | `apps/mobile/` 디렉터리에서 빌드 실행 |
| 살아있는 인물 노출 (오프라 윈프리 등) | `birth_year IS NULL` 조건 포함으로 birth_year 미입력 인물 통과 | 해당 조건 제거 |
